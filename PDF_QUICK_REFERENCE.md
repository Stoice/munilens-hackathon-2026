## Styling Hierarchy

### Headers & Sections
```
TITLE (H1)
━━━━━━━━━━━━
Large, bold, gray, with bottom border

SECTION HEADING (H2)
Small caps, blue, uppercase

Subsection (H3)
Standard formatting

Body Paragraph
Regular text with proper spacing and line height
```

### Content Elements
- **Lists**: Proper indentation with bullets
- **Numbers**: Sequential numbering (1, 2, 3, ...)
- **Quotes**: Left-aligned border with italics
- **Tables**: Bordered cells with header row
- **Emphasis**: Bold for important text

## Export Process

### User Flow
```
1. Click "Generate AI Insight Report"
   ↓
2. Wait for AI to analyze data (10-30 sec)
   ↓
3. Review report on screen
   ↓
4. Click "Export as PDF"
   ↓
5. PDF generated with professional formatting
   ↓
6. Browser automatically downloads file
   ↓
7. File appears in Downloads folder:
   MuniLens_Weekly_Insight_Report_2026-03-28_10-30.pdf
   ↓
8. Share, print, or archive as needed
```

## Troubleshooting Quick Guide

| Problem | Solution |
|---------|----------|
| PDF blank | Refresh page, try again |
| File too large | Normal for detailed reports |
| Text cut off | Check page breaks in PDF viewer |
| Images missing | Wait for content to load |
| Wrong formatting | Close & reopen PDF file |
| Can't download | Check browser downloads settings |
